## Quick commands ### Start Gateway ```bash lala gateway --port 18789 ``` ### Check status ```bash lala channels status ``` ### View logs ```bash lala logs --follow ``` ### Configuration location ``` ~/.lala/lala.json ``` ## Common tasks - **Add a new channel**: `lala channels login` - **Test connection**: `lala channels status --probe` - **Restart service**: `lala restart` - **Update configuration**: Edit `~/.lala/lala.json` then restart

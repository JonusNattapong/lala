## Quick fixes ### Installation issues - Check Node version: `node --version` (needs 22.16+) - Clear npm cache: `npm cache clean --force` - Reinstall: `npm uninstall -g lala && npm install -g lala@latest` ### Gateway won't start - Check configuration: `lala config validate` - View logs: `lala logs --follow` - Check ports: `lsof -i :18789` ### Channel connection problems - Test connection: `lala channels status --probe` - Re-authenticate: `lala channels login` - Check logs for specific channel errors ## Getting help - **GitHub Issues**: [Create an issue](https://github.com/JonusNattapong/lala/issues) - **Discord Community**: Join our Discord server - **Documentation**: [Full docs](/docs) ## Common error messages - `EADDRINUSE`: Port already in use - change port or kill existing process - `AUTH_FAILED`: Channel authentication failed - re-login with `lala channels login` - `CONFIG_INVALID`: Configuration error - run `lala config validate`

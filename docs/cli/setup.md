# `lala setup`

Initialize `~/.lala/lala.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
lala setup
lala setup --workspace ~/.lala/workspace
```

To run the wizard via setup:

```bash
lala setup --wizard
```
